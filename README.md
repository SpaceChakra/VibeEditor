# VibeEditor

VibeEditor is an intuitive visual 3D game editor for vibe coded games, built as a collaboration between SpaceChakra and Randroid's Dojo.

SpaceChakra built the original editor as part of his vibe coded game, Bone Cleaver. The origin and design intent are introduced in his video, [Vibe Coded Modeling/Animation Tools: Building a 90s Gore Character Editor Solo](https://youtu.be/YakJ-eGVQ-Y?si=rd1Ah_R51x0rrvqj).

The editor is designed around first-class support for destructive character and object behavior: slicing limbs off characters, dropping severed parts, and breaking pieces away from destructible 3D objects should be natural authoring workflows rather than afterthoughts. The current visual direction has PlayStation 1 era vibes, while leaving room for the graphics to improve as generative AI tooling gets better.

Randroid is extending the project into a more general-use editor. SpaceChakra has donated the first few models to help bootstrap that work.
